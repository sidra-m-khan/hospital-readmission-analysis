# Hospital Readmission Analysis (Diabetes Dataset)

## Project Overview
Analysis of hospital readmission among diabetic patients using SQL and Power BI. Focused on identifying factors that influence readmission rates.

## Tools Used
- SQL Server
- Power BI
- Excel

## Key Steps
- Data cleaning and preprocessing using SQL
- Business questions analysis
- Interactive dashboard creation in Power BI
